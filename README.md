## 📌 프로젝트 소개

**글결**은 **웹에서 드래그한 텍스트를 OpenAI API로 처리하여 문맥에 맞는 유의어 또는 대체 어휘를 추천**하는 **Google 확장 프로그램**이다.  
20대 PC 크롬 사용자를 주요 대상으로 하며, 글 작성 중 자연스럽게 어휘력을 향상시키고 글의 완성도를 높이는 데 도움을 준다.

## 💡 개발 배경

### 문제 인식

1. **글쓰기 수요 증가**
   - 한강 작가의 노벨상 수상 여파로 독서와 글쓰기에 대한 대중의 관심이 크게 증가했다.
   - 네이버 블로그 챌린지에서, 2023년에는 54만 명이 230만 건의 글을, 2024년에는 35만 명이 170만 건의 글을 작성하는 등 많은 사람이 글을 쓰고자 하는 의지를 가지고 있다. 특히 참여자의 90%가 30대 이하였다.

<br/>

2. **젊은 세대의 어휘력 부족**
   - 학생들이 '금일'을 '금요일'로, '고지식'을 '높은 지식'으로 오해하는 등 어휘력이 부족한 사례가 증가하고 있다.
   - 학생들이 다양한 감정이나 상황을 표현할 때, 풍부한 어휘 대신 유행어나 간단한 표현으로 대체하는 경향이 있다.
   - 학생들이 단어의 뜻을 설명하지 못한다.
   - 대학생이 작성한 글의 64%가 수준 미달이었다.
   - 직원의 국어 능력에 대한 평균 만족도 조사에서, 20대가 가장 낮은 점수를 기록했다.
  
<br/>

3. **대중의 문해력 저하 심각성 인식**
   - 사람인에서 진행한 '청년층 문해력 인식 설문조사'에서, 81.1%가 청년층의 문해력 저하 문제가 심각하다고 답했다.
   - 그 중 55.3%는 어휘력 저하가 곧 문해력 저하 문제로 이어진다고 대답했다.
   - 첫 질문의 81.1% 중 97.6%가 문해력 저하 문제를 해결하고자 하는 노력이 필요하다고 답했다.

<br/> 

### 솔루션

웹에서 글을 작성하는 20대 사람들에게, 글 작성 중 어휘 검색을 도와주는 서비스를 만들자.


1. **기존 서비스 한계**  
   - **네이버 국어사전:** 유의어 제공은 가능하지만, 여러 단어의 뜻을 비교하기 번거롭다.  
   - **낱말 서비스:** 네이버 국어사전보다 많은 양의 비슷한말/반대말/방언을 제공하지만, 여전히 한눈에 뜻을 비교하기ㅡ는 번거로우며, 대부분의 기능이 유료이다.  
   - **AI 기반 글쓰기 도구:** 문장 전체를 재작성하기 떄문에, 사용자가 생각할 기회를 주지 않는다.  
   - **Thesaurus Online:** 확장 프로그램 형태로 제공되지만, 사전의 기능보다도 가치가 떨어진다.  

<br/>

2. **글결의 차별성**
   - 크롬 확장 프로그램으로 개발해, 접근성을 높여 별도의 웹 사이트 접속 없이도 빠르고 편리하게 이용할 수 있도록 한다.
   - 단순히 단어와 그 유의어를 나열하지 않고, 문맥 기반 분석을 통해 작성 중인 문장에서 비슷하지만 다른 표현을 제시한다.
   - 제시한 표현의 구체적인 의미를 한눈에 비교할 수 있게 보여준다.
   - 문장 전체를 재작성하지 않고 사용자에게 다른 표현을 의미와 함께 추천해주기 때문에, 사용자가 능동적으로 의미를 이해하고 스스로 글을 수정하게 하여 어휘력 및 글쓰기 능력 향상에 도움을 줄 수 있다.
   - 체험용 무료 버전이 있다.

<br/>

## ✨ 주요 기능

1. **유의어 검색**  
   - 사용자가 드래그한 텍스트를 OpenAI API로 처리하여 적합한 유의어와 뜻을 반환한다.

<br/>

2. **문맥 기반 추천**  
   - 단순 단어 검색을 넘어, 문장에서 자연스러운 어휘 및 표현을 분석해 추천한다.

<br/>

3. **사용자 편의성**  
   - 별도의 웹사이트 접속 없이 Google 확장 프로그램으로 빠르게 검색이 가능하다.

<br/>

## 📈 성장 전략

### 서비스 개선
1. 전용 데이터베이스 구축
   - 유의어 및 단어 간 유사도 데이터를 포함한 전용 데이터베이스를 구축하여, 단순 검색 작업은 AI 호출 없이 데이터베이스로 처리한다. 이를 통해 처리 속도를 개선하고, API 호출 비용을 절감한다.

<br/>

2. 국립국어원 콘텐츠 연계를 통한 AI 개선
   - 국립국어워 콘텐츠 연계로 더 정확한 뜻풀이럴 제공하고, 국립국어원 콘텐츠를 학습시킨 AI 모델을 개발해 추천 성능을 강화하고 복잡한 표현의 검색 정확도를 개선한다.

### **수익 모델**
1. **사용자 플랜**
   - 무료 사용자 플랜
      - AI 이용 토큰 수 제한
      - 제한 초과 후 DB에서의 단순한 검색만 제공
      - 한 번에 검색 가능한 단어 수 제한
   - 프리미엄 사용자 플랜 *(월 3,000원)*
      - AI 이용 제한 해제
      - 한 번에 검색 가능한 단어 수 제한 대폭 증가

<br/>

2. **어휘 확장팩 판매**  
   - 특정 주제(예: 장르 소설, IT 전문 용어 등)와 난이도(예: 중등 수준)에 특화된 데이터베이스를 구축해 확장패긍로 판매한다.  
   - 사용자 맞춤형 서비스로 수익 다각화 실현한다.  

<br/>

3. **B2B 협업**  
   - 창작자 플랫폼, 글쓰기 플랫폼, 교육기관, 기업의 문서 작성 툴 등에 라이선스 판매.
   - 사용자들이 유의어 검색을 위해 다른 사이트로 이탈하지 않고 플랫폼 내에서 완료하도록 유도해 체류 시간을 늘리고, 더 나은 글 작성 경험을 제공한다.
   - 사용자 만족도를 높여 플랫폼의 재방문을 촉진한다.
